Please note this is currently an open beta, bugs are expected!

This is designed to be a web app to help people share their secrets. It has the ability to keep track of a whole bunch of secrets and also allows people to vote on secrets that they do or do not like, in order to help keep a positive vibe. Let's keep this a positive space!
If you don't like a secret, then down vote it. If you do like it, then feel free to up vote it. It helps the system pick which secrets you should see.
I plan to create user acounts in order to better keep track of what secrets you have already seen as well as your votes but that is a long way off
If you encounter an error, please contact me as soon as posible.
